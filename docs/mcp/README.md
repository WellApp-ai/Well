# Well MCP Server

Connect your AI assistant to Well financial data using the Model Context Protocol.

## What is MCP?

The **Model Context Protocol (MCP)** is an open standard that allows AI assistants like Claude, Cursor, and ChatGPT to securely access your Well data - invoices, companies, contacts, and more.

## Why Use Well MCP?

| Feature | Description |
|---------|-------------|
| **Natural Language Queries** | Ask "Show me unpaid invoices over 1000 EUR" and get instant answers |
| **Real-time Data** | Access your latest invoices, companies, and contacts directly |
| **Create Records** | Create companies and contacts through natural conversation |
| **Secure** | OAuth authentication keeps your data safe |

## Supported Clients

- VS Code (with MCP extension)
